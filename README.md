# ADPROCLUS

This package is an implementation of ADditive PROfile CLUStering (ADPROCLUS) in R. It can be used to obtain overlapping clustering models for object-by-variable data matrices.
### Dependencies

R version >= 3.1.0
```
gtools
NMFN
```
